Tutorial Penggunaan

Panduan penggunaan aplikasi untuk menampilkan chat, gift, dan event TikTok Live ke stream.

1. Install Aplikasi

Download aplikasi RVTik dari halaman download.

Jika file masih dalam bentuk zip, extract terlebih dahulu.

Jalankan file:

RVTik.exe

Jika muncul Windows Defender warning:

More Info → Run Anyway
2. Menghubungkan Akun TikTok

Isi informasi berikut pada dashboard:

Session

RV_XXXXX

Session digunakan untuk menghubungkan aplikasi dengan overlay.
* Session tidak dapat diubah, karena sudah build-in dalam penginstalan.

Username

amikruangvisual
3. Memulai Live Connection

Klik tombol:

START

Status akan berubah menjadi:

Connected

RVTik akan mulai membaca event dari TikTok Live seperti:

  • Chat
  • Gift
  • Join
  • Follow
4. Menambahkan Overlay ke OBS

Buka menu Overlay Links di dashboard.

Copy link overlay yang ingin digunakan.

Di OBS:

  • Sources
  • Add
  • Browser Source
  • Paste link overlay

Ukuran yang disarankan:

1920 x 1080
5. Menggunakan Dock Panel

Dock adalah panel kontrol untuk melihat event live secara realtime.

Fungsi dock:

  • Melihat gift masuk
  • Melihat chat penonton
  • Monitoring event live

Buka dock dengan tombol:

DOCK
6. Agar Aplikasi Lebih Ringan

Untuk performa terbaik saat streaming:

  • Minimize Dashboard RVTik
  • Biarkan Dock tetap aktif
  • Aplikasi akan berjalan di background
Tips: Cara ini membuat penggunaan CPU lebih ringan saat streaming.
7. Membuka Dashboard Lagi

Jika dashboard diminimize, RVTik akan tetap berjalan di background.

Buka kembali melalui:

System Tray Windows (pojok kanan bawah dekat jam)

Caranya:

  • Cari icon RVTik
  • Double click icon
  • Dashboard akan muncul kembali
8. Menghentikan Koneksi

Jika live sudah selesai klik tombol:

STOP

Koneksi dengan TikTok Live akan dihentikan.

9. Menambahkan Overlay ke OBS

Buka menu Overlay Links di dashboard RVTik.

Copy link overlay yang ingin digunakan (Chat / Gift / All-in-One).

Langkah di OBS:

  • Buka OBS Studio
  • Pada bagian Sources klik tombol +
  • Pilih Browser Source
  • Beri nama (contoh: RVTik Overlay)
  • Klik OK

Masukkan Link:

  • Paste link overlay ke kolom URL
  • Atur ukuran:
Width: 1920 | Height: 1080

Klik OK, overlay akan langsung muncul di layar OBS.

Tips: Pastikan RVTik sudah dalam status Connected agar overlay menampilkan data realtime.
10. Menambahkan Overlay ke TikTok Live Studio

Selain OBS, kamu juga bisa menggunakan RVTik di TikTok Live Studio.

Langkah-langkah:

  • Buka aplikasi TikTok Live Studio
  • Masuk ke scene live kamu
  • Klik tombol Add Source (+)
  • Pilih Browser atau Web Page

Masukkan Overlay:

  • Paste link overlay dari dashboard RVTik
  • Atur ukuran sesuai layar:
1920 x 1080

Klik Confirm / Add, overlay akan langsung tampil di live.

Jika overlay tidak muncul:

  • Pastikan status RVTik sudah Connected
  • Coba refresh source
  • Pastikan link overlay benar
Tips: TikTok Live Studio kadang butuh refresh manual agar overlay update realtime.
